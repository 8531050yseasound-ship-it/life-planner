📥 Instagramの保存済み投稿を取り込む

Instagramには保存済みコレクションを外部アプリへ自動連携する仕組みがないため、 公式の「データをダウンロード」機能で書き出したファイルを取り込みます。

  1. Instagramアプリ → プロフィール → ☰ → アクティビティ（またはアカウントセンター）→ 個人データをダウンロード
  2. 「一部の情報」→ 保存済み にチェック → 形式は JSON を選んでリクエスト
  3. 届いたZIP（または展開した saved_collections.json / saved_posts.json）を下で選択

保存元の投稿URLとアカウント名だけが取り込まれます（写真や場所名はエクスポートに含まれないため、名前は後から編集してください）。取り込み済みのURLは重複しません。

💡 Androidのちょい足し: このアプリをホーム画面に追加すると、Instagramの投稿の「共有」→ Life Planner で1件ずつ直接追加できます。

🏦 銀行・カード・証券の明細を取り込む

楽天カード・楽天銀行・楽天証券・ゆうちょ銀行には、外部アプリが残高や明細を自動取得できる公式の仕組み（一般個人向けAPI）がありません。 ログイン情報を使った自動取得はセキュリティ上おこなえないため、各社サイトからご自身でダウンロードした明細CSVを取り込みます。

  1. 楽天カード：PC版 楽天e-NAVI → ご利用明細 → 「明細コピー用 Excel(CSV)」でダウンロード
  2. 楽天銀行：PCでログイン → 入出金明細 → 期間を指定 → 「CSV形式でダウンロード」
  3. ゆうちょ銀行：ゆうちょダイレクト → 入出金明細照会 → 「データダウンロード（CSV形式）」（ZIPのままでOK）
  4. 楽天証券：入出金・振替履歴（またはマネーブリッジ明細）→ CSVダウンロード

取り込み先は現在のユーザー「」です。出金は支出、入金は収入として記録され、内容から自動でジャンル分けします（あとで各行を修正できます）。同じ明細の二重取り込みは自動でスキップします。文字コード（Shift_JIS/UTF-8）は自動判定します。
楽天銀行の「楽天カードサービス」（カード利用分の引き落とし）は、楽天カード明細と二重計上になるため自動で除外します。
